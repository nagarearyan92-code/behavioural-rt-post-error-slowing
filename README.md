# Behavioural Reaction-Time Analysis: Post-Error Slowing

This project demonstrates a basic behavioural data analysis workflow using Python, pandas and Jupyter Notebook.

The analysis uses a small simulated trial-level dataset to demonstrate how to:

- clean unrealistic reaction-time values
- create previous-trial labels within each participant
- calculate mean reaction time after correct and error trials
- calculate post-error slowing overall
- calculate participant-level post-error slowing

## Skills demonstrated

- Python/Jupyter Notebook
- pandas DataFrames
- reaction-time cleaning
- participant-level behavioural analysis
- groupby operations
- previous-trial labelling
- basic research-style interpretation

## Note

This project uses simulated data for learning and demonstration purposes. The results are not intended for statistical inference.
